Features

Everything in one calm board

No spreadsheets, no sticky notes. Trackix turns the chaos of half-finished side-projects into clean, draggable cards.

🔍

Auto-detects projects

Pick a folder and Trackix finds every project inside it — by git repo, package.json, requirements.txt and more.

🧠

Smart status guess

A built-in heuristic reads README, deploy configs, tests and TODOs to suggest finished / in-progress / dropped.

🛠️

Tools & hosting

See the stack each project uses and where it's deployed — Vercel, Netlify, GitHub, Docker and others.

✦

Optional local AI

Flip on Ollama for sharper status calls. It runs on your machine — no API keys, no data leaving your device.

🪄

Drag & drop

Move cards between columns or drag them to the bin. Smooth animations, refreshable status, the works.

🔒

100% local & private

Trackix only ever looks inside the folder you pick — never your whole disk or system files. Nothing is uploaded.

🔥

Revival Ritual

Resurrect dead projects. Trackix reads what remains, tells you why it died, and forges a step-by-step path back to shipped.

👁️

The Oracle

Can't pick what to work on? Consult the Oracle — it weighs momentum and distance-to-done, then reveals the one project that deserves you today.

🌌

The Cosmos

Your work as a living night sky. Every project becomes a star — size is completion, pulse is momentum, constellations connect shared stacks. No tracker has ever looked like this.

How it works

Up and running in 30 seconds

1

Pick a folder

Point Trackix at your Downloads or projects folder. It scans only what's inside.

2

Review & add

It lists every project with a suggested status. Tick the ones you want to track.

3

Stay on top

Drag cards between In Progress, Finished and Dropped. Hit refresh to re-scan anytime.
