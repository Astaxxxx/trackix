Trackix — drop-in assets
========================

Put image files here to personalise your copy of Trackix. They are picked up
automatically (no code changes needed). If a file is missing, the app falls
back to its built-in vector art.

  mascot.png       The chibi mascot shown on the empty screen + faint in the
                   background. Use a square-ish PNG with a transparent
                   background (e.g. a chibi sticker you like). ~512x512+.

  astax-logo.png   Your Astax Labs logo, shown in the footer. Small, square,
                   transparent background works best. ~64x64+.

Note: character art / stickers from the internet are usually copyrighted. Using
one in your own local copy is your call — but don't ship someone else's
character art if you distribute or sell Trackix. The built-in vector mascot is
original and safe to ship.
